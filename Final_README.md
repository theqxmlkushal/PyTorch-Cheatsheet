# 🔥 PyTorch Cheat Sheet

A **clean, interactive PyTorch cheat sheet** built with **React + Vite** that provides quick reference snippets for deep learning development.

It covers everything from **basic tensors to advanced PyTorch features**, making it useful for **students, ML engineers, and researchers** working with **PyTorch**.

---

🌐 [https://pytorch-cheatsheet.vercel.app](https://pytorch-cheatsheet.vercel.app)

# 📚 Features

✔ Quick reference for common PyTorch APIs
✔ Copy-to-clipboard code snippets
✔ Organized by concept and workflow
✔ Dark developer-friendly UI
✔ Covers beginner → advanced PyTorch usage

Sections include:

* **Tensors**
* **Operations**
* **Autograd**
* **nn.Module**
* **Loss Functions**
* **Optimizers**
* **Training Loops**
* **Advanced Topics**

PyTorch is widely used for research and production deep-learning models because it provides flexible tensor computation and automatic differentiation tools. ([GitHub][1])

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/theqxmlkushal/PyTorch-Cheatsheet.git
```

Move into the project directory:

```bash
cd PyTorch-Cheatsheet
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Open:

```
http://localhost:5173
```

---

# 🚀 Deployment

The project can be deployed easily using **Vercel** or **Netlify**.

Build the project:

```bash
npm run build
```

This generates the **dist/** folder which can be deployed as a static site.
